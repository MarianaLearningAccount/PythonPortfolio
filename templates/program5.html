This program checks email validity against the following criteria:

  1. It contains a single "@" symbol.
  2. It contains at least one dot (.) after the "@" symbol.
  3. The username name (before "@" symbol) has at leats a letter or a number.
  4. The mail server name (after "@" symbol) has at leats a letter or a number.
  5. The domain name (after the last dot) has at least two characters.
